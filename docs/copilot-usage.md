# GitHub Copilot Usage

GitHub Copilot was used throughout the SnapSplit development process.

Copilot helped with:
- creating the Expo React Native mobile app structure;
- building screens and navigation;
- improving TypeScript types;
- creating receipt review and split logic;
- debugging Expo and React Native errors;
- creating the Node.js API backend;
- integrating Azure AI Document Intelligence;
- improving UX based on testing;
- writing documentation and project structure.

Copilot Agent mode was also used to apply multi-file changes across the mobile app and backend API.
